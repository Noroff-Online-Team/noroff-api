# Noroff API docs

This is the documentation for the Noroff API. It's available at [docs.noroff.dev](https://docs.noroff.dev).

It's built with [Fumadocs](https://fumadocs.dev).

## Running the docs

From root you can specify the app with `-F docs...` flag

```bash
pnpm dev -F docs...
```

## Making changes

To make changes to the docs, you can edit the files in the `content` directory. The files are written in MDX format.
